🎬 Netflix Clone (Frontend)

A responsive Netflix India landing page clone built using HTML and CSS. This project focuses on recreating the UI design of Netflix’s homepage with modern layout techniques and responsive design.<br>

📌 Features<br>
🔥 Clean and modern Netflix-style UI<br>
📱 Fully responsive design (mobile + desktop)<br>
🎥 Video overlay inside device frames<br>
🎯 Hero section with call-to-action<br>
❓ FAQ section with hover effects<br>
📂 Structured footer layout<br>
🌐 Google Fonts integration<br>
<br>
<ul>
<h4>🛠️ Tech Stack</h4>
<li>HTML5</li>
<li>CSS3 (Flexbox + Grid)</li>
<li>Responsive Design (Media Queries)</li>
</ul>
